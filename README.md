# AI_note_making
An AI-powered offline note-making app that transforms boring paragraphs into concise, structured notes. Perfect for students, professionals, or anyone who wants to quickly summarize and organize information without an internet connection.

Key Features:
📝 Convert long paragraphs into clear, concise notes
⚡ Works completely offline
🔒 Privacy-focused: no data leaves your device
📚 Ideal for study, work, or personal note-taking
